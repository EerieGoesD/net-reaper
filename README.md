# Net Reaper

Download manager. Windows, macOS, Linux. Tauri + Rust.

## Build

```
npm install
npm run dev
```

```
npm run build:nsis      # Windows
npm run build:dmg       # macOS
npm run build:appimage  # Linux
```

Requires [Rust](https://rustup.rs/) and [Node.js](https://nodejs.org/).
